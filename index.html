What is it?

SnapMotion is a free, open source (MIT License) jQuery plugin. It is easily integrated into your website.

Compatibility

Firefox 3+, Chrome 5+, Safari 3+, Opera 9+, IE6+.

View our demonstrations:

Features

This plugin was built to make creating a slideshow on your website extremely simple and painless. By including the few files provided in this package, you'll be up and running within minutes.

  • Auto-sliding
  • Image titles & captions
  • Flexible configuration

Documentation

Firstly, you'll need to reference the SnapMotion JavaScript and CSS files within your <head> tag.

						<script src="js/jquery.snapmotion.js"></script>
						<link rel="stylesheet" href="css/snapmotion.css" type="text/css" />

Create your slides using a <ul> with a <li> for each slide. Each <li> should include the slide <img /> and a <span> tag containing any text you'd like to display on the slide.

For example:

						<ul id="slideshow">

    Finally, call the SnapMotion() method on your slideshow. For example:

    						$("ul#slideshow").SnapMotion();

    Optional parameters include:

    • slideWidth (int, the width of a slide)
    • slideHeight (int, the height of a slide)
    • showCaption (boolean, whether to show a caption)
    • captionWidth (int, the width of the caption area)
    • captionHeight (int, the height of the caption area)
    • showControls (boolean, whether to show next/previous controls)
    • controlIndentX (int, the indentation of the 'controls' for the slideshow, in pixels, from the left of each slide)
    • controlIndentY (int, the indentation of the 'controls' for the slideshow, in pixels, from the top of each slide)
    • animationLength (int, the length, in milliseconds, of each slide transition)
    • delay (int, the length, in milliseconds, between each slide transition)
    • transition (string, slideX (default), slideY, fade)
    						<script type="text/javascript">
							$(document).ready(function(){
								$("ul#slideshow").SnapMotion({
									'slideWidth'		: 960,
									'controlIndent'		: 640,
									'animationLength'	: 375,
									'delay'				: 7500,
									'transition'		: 'slideY'
								});
							});
